Project 1

This Identity Manager project is a Python-based application designed to efficiently manage personal information. Built with a user-friendly interface using Tkinter, it allows users to add, search, remove, and display individuals' data such as name, SSN, phone number, address, and more. The application ensures data validation, including proper formatting for phone numbers, SSNs, and other fields, to maintain data accuracy. The data is securely stored in a JSON file for easy access and modification. It's ideal for small-scale data management and serves as an excellent example of how Python can handle everyday tasks through simple, interactive applications.
